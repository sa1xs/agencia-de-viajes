# Agencia de Viajes Responsive

Proyecto de una página web responsive para una agencia de viajes, desarrollado con **HTML** y **CSS** como parte de una tarea académica del curso de Diseño y Programación Web.

## Descripción

Este proyecto consiste en el diseño y desarrollo de un sitio web completo para una agencia de viajes ficticia. La página fue creada con una estructura clara y semántica, incorporando secciones informativas, tarjetas de destinos, ofertas destacadas, imágenes representativas y un diseño adaptable a diferentes tamaños de pantalla.

El objetivo principal fue aplicar conceptos de maquetación web, estilos visuales y diseño responsive, garantizando una experiencia de usuario ordenada y funcional en dispositivos móviles, tabletas y escritorio.

## Características principales

- Estructura semántica con `header`, `nav`, `main` y `footer`
- Menú de navegación con enlaces principales
- Sección de bienvenida con imagen destacada y botón de acción
- Tarjetas de destinos turísticos
- Sección de ofertas especiales
- Diseño responsive para celular, tablet y escritorio
- Uso de **Flexbox** y/o **Grid**
- Implementación de **media queries**
- Estilos desarrollados únicamente con CSS propio

## Tecnologías utilizadas

- HTML5
- CSS3
